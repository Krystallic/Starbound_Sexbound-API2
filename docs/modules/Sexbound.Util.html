Sexbound.Util.sendMessage (entityId, name, args, wait)
Handles sending a message to a specified entity.

Parameters:

  • entityId String: A remote entity id or unique id.
  • name String: The name of the message to send the remote entity.
  • args Table: Arguments to send to th remote entity.
  • wait Boolean: true = wait for response before sending again. false = send without waiting

Usage:

  • Sexbound.Util.sendMessage(entityId, "sexbound-node-init", {}, false)
  • Sexbound.Util.sendMessage(uniqueId, "sexbound-node-init", {}, false)
