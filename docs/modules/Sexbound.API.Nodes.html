Sexbound.API.Nodes.addNode (tilePosition, sitPosition)
Add a new node object to the current world.

Parameters:

  • tilePosition A Vec2I position to place the 8 x 8 pixel node. It is placed relative to the imagePosition of this object.
  • sitPosition A Vec2I position to place the character who sits in this object.

Usage:

    Sexbound.API.Nodes.addNode({0, 0}, {0, 20})
